# CV Models Directory

This directory contains all computer vision models used by the proctoring system.

## Models

- **DeepFace Models**: Automatically downloaded by DeepFace library (stored in user home directory)
- **Custom Models**: Place any custom models (e.g., phone detection models) here

## Structure
```
cv_models/
├── phone_detector/       # Phone detection model (future)
├── object_detector/      # Generic object detector (future)
└── README.md            # This file
```
